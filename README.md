# Contact Manager

**Contact Manager** is a .NET application for managing contacts by uploading CSV files and storing the data in a MS SQL database.

The project allows you to:
- Upload a CSV file with contacts (e.g., Name, Date of Birth, Salary, etc.).
- Store contacts in an MS SQL database.
- View and manage the data through a simple web interface.
